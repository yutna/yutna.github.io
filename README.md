# Yutna's website

Welcome to my personal website repository.

## Prerequisites

Ensure that these tools are installed on your local machine prior to running
this project.

- [Node.js (v20.9.0)][1]
- [Visual Studio Code][2]
- [Astro support for Visual Studio Code][3]

## Getting started

This website is built on top of the [Astro][4] framework. There is no fancy
setup required. You can simply run this project in three easy steps as outlined
below.

- Clone the project repository to your local machine

  ```bash
  git clone git@github.com:yutna/yutna.github.io.git
  ```

- Enter the project directory and install all dependencies.

  ```bash
  cd yutna.github.io && npm install
  ```

- To start the local development server, run the following command

  ```bash
  npm run dev
  ```

## Commands

Here are all the listed commands.

| Command                   | Action                                           |
| :------------------------ | :----------------------------------------------- |
| `npm run astro ...`       | Run CLI commands like `astro add`, `astro check` |
| `npm run astro -- --help` | Get help using the Astro CLI                     |
| `npm run build`           | Build your production site to `./dist/`          |
| `npm run dev`             | Starts local dev server at `localhost:4321`      |
| `npm run format`          | Format code using Prettier                       |
| `npm run preview`         | Preview your build locally, before deploying     |
